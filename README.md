# Val Town (val-town)

Val Town is a social coding platform where JavaScript and TypeScript programs called "vals" run instantly as HTTP endpoints, scheduled cron jobs, or email handlers — no infrastructure required. It serves developers who want to write, share, and deploy small server-side scripts directly in the browser. Val Town exposes a versioned REST API (v1/v2/v3) covering vals management, blob storage, SQLite, email, telemetry, and OAuth connections, along with an official TypeScript SDK and a hosted MCP server for LLM integrations.

## APIs

- **Val Town REST API** — Full REST API (v1/v2/v3) for managing vals, blob storage, SQLite databases, email, telemetry, users, and OAuth connections.
  - Docs: [https://docs.val.town/reference/api/](https://docs.val.town/reference/api/)
  - OpenAPI: [https://api.val.town/openapi.json](https://api.val.town/openapi.json)
- **Val Town TypeScript SDK** — Official @valtown/sdk npm package autogenerated from the OpenAPI spec via Stainless; supports Node.js 20+, Deno, Bun, and Cloudflare Workers.
  - Docs: [https://docs.val.town/sdk/](https://docs.val.town/sdk/)
  - GitHub: [https://github.com/val-town/sdk](https://github.com/val-town/sdk)
- **Val Town MCP Server** — Hosted Model Context Protocol server at api.val.town/v3/mcp for LLM and AI agent integrations.
  - Docs: [https://docs.val.town/](https://docs.val.town/)
